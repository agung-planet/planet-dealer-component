## How to install
### Jitpack
Type this on your build.gradle (project)
```sh
allprojects {
    repositories {
      ...
        maven { url "https://jitpack.io" }
    }
}
```

How to use :
```sh
implementation "com.github.oddy-dev.planet-dealer-component:dealer-component:master-SNAPSHOT"
```
